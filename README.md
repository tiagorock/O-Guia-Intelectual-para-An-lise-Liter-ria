📚 O Guia Intelectual para Análise Literária
Este projeto utiliza o NotebookLM como um laboratório de inteligência artificial para elevar o estudo de obras clássicas e contemporâneas. O objetivo é transformar a leitura passiva em uma análise crítica multimodal, integrando síntese textual, debate em áudio e estruturação de dados.
🚀 Destaques do Repositório
Curadoria Diversificada: Análise de 23 fontes, abrangendo desde o Realismo de Machado de Assis e o Naturalismo de Aluísio Azevedo até ficções históricas como Sonata em Auschwitz.
Ecossistema de Artefatos: Produção de podcasts de debate (Audio Overviews), relatórios persuasivos para blogs e tabelas comparativas de personagens.
Metodologia Grounded: Todo o conteúdo é rigorosamente ancorado nos textos originais, garantindo análises fiéis ao estilo e contexto dos autores.
Ferramentas Reutilizáveis: Inclui glossários de conceitos (como "Banalidade do Mal" e "Determinismo") e prompts estruturados para novas análises.
Este projeto demonstra como a IA pode atuar como um "parceiro de pensamento", aprofundando a compreensão sobre a natureza humana e a estrutura narrativa.
